# piniaApp Demo

## Dev
Pasos

1. Clone the project

2. Run 
```
yarn install
npm install 
```

3. Raise the backend server
```
npm run backend-server
yarn backend-server
```

4. Run the Vue.js application
```
npm run dev
yarn dev
```
